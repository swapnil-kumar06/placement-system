# 🚀 Placement & Internship Tracking System

A full-stack web application designed to manage student placements, internships, and company eligibility efficiently.

---

## 📌 Features

- 👨‍🎓 Add and manage student records  
- 🏢 Add and manage company details  
- ✅ Track student placement status  
- 📊 Check eligibility based on CGPA  
- 🔄 Automatic status update using SQL Trigger  
- 📋 View placed students using SQL View  
- ⚙️ Stored Procedure for eligibility checking  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js (Express)  
- **Database:** MySQL  

---

## ⚙️ Database Concepts Used

- ✔ Primary Key & Foreign Key  
- ✔ Normalization (up to 3NF)  
- ✔ Trigger (Auto update placement status)  
- ✔ View (Placed Students)  
- ✔ Stored Procedure (Eligibility Check)
